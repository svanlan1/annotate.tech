Annotate! Extension Changelog

September 29th, 2016

Version .0.6.40
  • Blank annotations can no longer be saved to the server.

August 11th, 2016

Version .0.6.37
Official Beta release 1.01
  • Notes are now saved to the database. Notes are different than text on the screen. To add a note, right click on a placed pin.
  • Annotate is now touch friendly! Use the extension on your touch screen device (must use in desktop version of Google Chrome - works great on Surface Pro 4)

August 9th, 2016

Version .0.6.36
Official Beta release 1.0
  • Users are now able to save annotations straight to their account at annotate.tech
  • Without creating an account, annotations are stored to the browser. WARNING these annotations will be removed by clearing the browser cache.
  • Updated the options page. Log in to your account by visiting the 'Options' page. Complete UI revamp

August 1st, 2016

Version .0.6.35
  • Users have the ability to create boxes on the page
  • Users can place pins anywhere on the page
  • Fixed issues with adding a text annotation to the page
  • The ability to 'Save' annotations has been added. Currently this feature saves the pins, boxes, and text. It does not save 'Notes'.
    Saving 'Notes' will be available in version 0.6.35
  • Display previous annotations by clicking on the 'Save' icon in the upper left-hand corner of the Annotate extension
  • Version 0.6.36 will include the ability to login and view saved annotations at annotate.tech

July 3rd, 2016

Version .0.6
  • First official release of Annotate! extension on the Chrome Web Store
  • Place pins, draw boxes, and add text to the screen
  • Add custom recommendations. Choose from HTML, Accessibility, and Blank.
  • All recommendations are editable and saveable
About svA11y

Annotate! was created and is maintained by Shea VanLaningham. svA11y.com is a website dedicated to providing quality Web Accessibility and Section 508 consultation and remediation. Annotate! was created to assist users in making Accessibility notations, but quickly grew into something much bigger and better!
